# Romania Tech Events · 2026

A single-page, dependency-free website listing the major upcoming **technology events in Romania** in 2026 — conferences, hackathons and summits.

🔗 **Live site:** https://events.alexandrudan.com

## Features

- **Upcoming-event rows** — click any event to expand full details (description, facts, topics).
- **Live countdown** to the next event.
- **Filters** — search, city chips, category chips (Conference / Hackathon / Academic / Festival), "Next 30 days" and "Saved" toggles.
- **Save your shortlist** (stored locally in your browser).
- **Calendar sync** — add any event to Google Calendar / Outlook, or download `.ics` (single, all, or just your saved agenda).
- **Shareable links** — filters sync to the URL; copy a link to any filtered view.
- **Print / PDF** friendly, keyboard shortcuts (`/` to search), accessible and mobile-responsive.

## Tech

Plain HTML/CSS/JavaScript in one file (`index.html`). No build step, no dependencies.

## Notes

Event dates are compiled from official event sites and the [dev.events](https://dev.events/EU/RO/tech) aggregator. **Always confirm dates on the official event sites before booking travel** — some are sourced from aggregators and may change.
